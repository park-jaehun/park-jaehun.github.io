# 1. TO_CHAR
TO_CHAR는 숫자 또는 날짜를 문자타입으로 변환시켜주는 함수입니다.
```
SELECT 기준일자
       ,TO_CHAR(영업일수)
       ,TO_CHAR(기준일자, 'YYYYMMDD')
FROM 기준일자정보
```
